## 🌌 全局背景动画

### 动画方案：浮动粒子网格

- **Light 模式**：极浅灰色粒子 `#dddddd`，连线透明度 `0.3`，几乎隐形，若隐若现
- **Dark 模式**：白色粒子 `rgba(255,255,255,0.15)`，连线透明度 `0.15`，星空感
- **粒子数量**：60~80 个，不密集
- **运动速度**：极慢，`0.3~0.5px/frame`，感觉像「呼吸」
- **鼠标交互**：鼠标靠近时粒子轻微排斥，产生涟漪感
- **实现**：`Canvas API` 或 `tsParticles`（轻量）

```
效果关键词：「你感觉到它在动，但不会被它打扰」
```

### 全局微动画规范

| 位置 | 动画 | 参数 |
|------|------|------|
| 页面载入 | 内容从下淡入 | `translateY(10px)→0, opacity 0→1, 0.4s` |
| 链接悬停 | 下划线从左展开 | `width 0→100%, 0.2s ease` |
| 卡片悬停 | 轻微上浮 | `translateY(-3px), 0.2s ease` |
| 导航切换 | 内容淡出淡入 | `opacity, 0.2s` |
| 模式切换 | 背景色渐变 | `0.4s ease` |
| 滚动视差 | 背景粒子层轻微偏移 | `parallax ratio: 0.3` |

## 🐱 数字宠物小猫系统

### 形象设计
- 基于**真实猫咪照片**提取，转为**矢量线条风格**（保留毛色特征，简化细节）
- 尺寸：`80×80px`，固定在页面**右下角**
- 动画帧数：`6~8帧` 循环，CSS Sprite 或 Lottie 实现

### 状态机设计

```
默认状态
├── 🐱 idle        坐着，尾巴左右摆动（循环）
├── 😴 sleeping    趴着，胸口起伏（长时间无操作后触发）
├── 🐾 walking     在底部边缘小跑（随机触发，5~10s）
└── 👀 curious     抬头看，耳朵竖起（鼠标停留在附近时）

导航交互状态（选中菜单时触发）
├── 🏠 首页        小猫伸懒腰，打哈欠
├── 📝 博客        小猫戴上小眼镜，翻书动作
├── 🎨 项目        小猫爪子比划，像在敲键盘
├── 🔧 工具        小猫叼着扳手走过来
└── 🤝 关于        小猫挥爪打招呼

点击交互状态
├── 单击           弹出对话框，小猫说话
├── 双击           小猫翻滚一圈
└── 长按           小猫不高兴，甩尾巴
```

### 💬 对话框系统

点击小猫后弹出气泡对话框：

```
┌─────────────────────────┐
│  喵～ 你好！             │
│                         │
│  今天想聊点什么？        │
│  ┌──────────────────┐   │
│  │ 输入消息...       │   │
│  └──────────────────┘   │
│  [发送]                  │
└─────────────────────────┘
     🐱
```

**对话框设计规范**：
- 背景：`--bg` 色 + `1px border`，无圆角（保持极简风）
- 字体：等宽字体，营造「终端感」
- 小猫回复：接入 LLM API，以猫咪口吻回答（偶尔夹杂「喵」）
- 对话框可拖拽移动
- 关闭方式：点击空白处或按 `ESC`

**预设对话场景**：
```
用户进入博客页 → 小猫主动说：「喵～ 要看什么文章？」
用户停留 >3min → 小猫说：「还在看呢，要不要喝杯水？」
深夜访问(22:00+) → 小猫打哈欠：「这么晚了还在卷...」
切换到 Dark 模式 → 小猫说：「黑夜模式，我最喜欢了 🌙」
```

## 📱 页面 UI 描述

### 🏠 首页
黑白粒子背景浮动，内容单列居中，层层淡入载入。顶部导航右侧：语言切换（EN/中）+ 模式切换图标。姓名等宽大字，副标题 muted 色。社交链接纯文字横排，悬停下划线展开动画。项目卡片细线边框，悬停上浮。右下角小猫 idle 状态，尾巴摆动。

### 📝 博客页
粒子背景降低透明度（阅读场景，减少干扰）。文章列表纯行式，年份分组。**小猫切换为戴眼镜状态**，在底部缓慢踱步。

### 🎨 项目页
两列卡片网格，细线边框。**小猫切换为敲键盘状态**，偶尔抬头看屏幕。卡片载入时交错淡入（`stagger: 0.1s`）。

### 🔧 工具页
与项目页布局相同。**小猫叼着工具走来**，停在页面中央做个动作后归位。工具卡片右上角绿点（唯一非黑白色）表示「可用」。

### 🤝 关于页
纯文章布局，行高宽松。**小猫挥爪打招呼**，然后坐在文字旁边陪读。`/now` 模块虚线框包裹。技术栈等宽字体横排。

## 🛠️ 技术实现

```
背景动画    →  tsParticles（轻量，3kb gzip）
小猫动画    →  Lottie（矢量动画，设计师出图）
             或 CSS Sprite（像素风备选）
对话系统    →  OpenAI API / 本地小模型
多语言      →  i18next 或 next-intl
明暗模式    →  CSS Variables + localStorage
框架        →  Next.js 15 + Tailwind
```

## ⚠️ 克制原则

```
✅ 背景粒子：存在但不抢戏
✅ 小猫动画：有趣但不打扰阅读
✅ 对话框：主动触发，不自动弹出骚扰
✅ 导航联动：小猫反应在 1s 内完成，不拖沓
❌ 不在文章正文区域叠加动画
❌ 不让小猫遮挡核心内容
❌ 不自动播放声音
```
